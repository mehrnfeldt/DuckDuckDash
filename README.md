DuckDuckDash
DuckDuckDash is a lightweight TypeScript browser game built with Vite.
The project uses a simple, modular structure that makes it easy to expand gameplay and UI components.

Features
Fast Vite dev server and build system

Modular TypeScript code

Clean, customizable UI

Easy to extend with new mechanics
